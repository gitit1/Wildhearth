# Wildhearth

Top-down 2D life-sim / farm game. All graphics are drawn in code — no asset files.

## Run
```bash
npm install
npm run dev
```
Move: WASD / arrows / touch-drag. Act: E / round button (mobile).
First loop: fish at the pond, sell at the stall, watch the coins persist.

See CLAUDE.md (agent guide), docs/VISION.md (design), docs/ROADMAP_MVP.md
(current build order), docs/ROADMAP_EXPANSION.md (everything after).
